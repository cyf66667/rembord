📖 单词速记系统 (Vocabulary Mastery App)
这是一个基于现代 Web 技术构建的模块化背单词学习平台。旨在通过科学的任务分配、智能的复习算法以及闭环的默写反馈，帮助学习者高效掌握四、六级及雅思等核心词汇。

🚀 架构设计
本项目采用功能模块化 (Feature-based Architecture) 设计，将 UI 骨架、逻辑中枢与数据层深度解耦，确保了代码的可维护性与扩展性。

✨ 已实现的核心功能
1. 智能词库管理系统
真实数据录入：支持加载本地 JSON 格式的真实词库（含四、六级、雅思等），解析音标、词义等完整数据。

自定义学习计划：用户通过设定开始与结束日期，系统自动平摊每日学习任务量，实现节奏化学习。

全量乱序洗牌：采用 Fisher-Yates 高效洗牌算法，确保每次生成的任务顺序随机，打破记忆惯性。

2. 交互式学习界面
平铺列表视图：摒弃传统翻页式操作，采用现代卡片式平铺列表，支持快速滑动浏览。

掌握程度评估：提供“已认识”、“熟悉”、“不熟悉”三级分类，精准定位学习盲点。

无打扰专注模式：一键进入沉浸式布局，隐藏冗余导航，最大限度减少干扰。

3. 反馈闭环与默写测试
智能生词本：将评价为“不熟悉”的词汇按日期归档，支持查看历史生词记录，便于查漏补缺。

多模式默写测试：

题库组合：自动提取当日学习任务，并从历史词库中抽取 10 个随机词构建测试池。

双向测试：支持“掩盖中文（英译汉）”与“掩盖英文（汉译英）”两种模式，深度检验拼写与词义。

💡 快速上手
环境准备：确保系统中已安装 Python。

启动服务器：
在项目根目录下打开终端，运行：

Bash
python -m http.server 8000
开始使用：
打开浏览器访问 http://localhost:8000 即可启动应用。
