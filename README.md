# Rembord 背单词

一个基于浏览器端的背单词应用，支持多种词库、专注模式、听写练习、能量花园、学习日历等功能。后端使用 Flask + MySQL 存储用户账户信息、头像及登录状态。

在线预览：https://cyf66667.github.io/rembord（纯前端演示版，完整功能需本地部署后端）

## 功能特性

### 核心学习
- **多词库支持**：四级核心词汇、六级高频词汇、雅思红宝书、考研必背词汇
- **每日任务**：根据设置的背诵起止日期自动计算每日背诵量
- **单词卡片**：展示英文、音标、中文释义，单击发音、双击跳转有道词典查看详细释义与例句
- **生词本**：自动按日期归档标记为"不熟悉"的单词，随时复习

### 专注模式
- 全屏沉浸式卡片背单词
- 点击卡片翻转查看中文释义
- 支持"认识 / 不认识"快速自评，不认识的词自动加入生词本

### 听写模式
- 系统语音朗读英文单词（使用 Web Speech API）
- 用户输入听到的单词，实时判断正误
- 支持从今日已学单词或任意词库中选题

### 默写测试
- 从今日已学单词 + 历史随机词汇中出题
- 支持两种模式：看英文写中文 / 看中文写英文
- 自我核对，不强制判分

### 能量花园
- 背单词获得水滴奖励
- 消耗水滴播种、浇水，培育植物成长
- 植物经历种子 -> 发芽 -> 幼苗 -> 开花四个阶段
- 开花后可收获，返还水滴并获额外奖励

### 学习日历
- 右上角日历图标查看学习记录
- 有学习记录的日期会显示绿色高亮
- 点击日期可查看当日学习的所有单词

### 用户账户系统
- **注册 / 登录**：使用手机号 + 密码，数据持久化到 MySQL
- **记住登录状态**：基于 Flask Session，刷新页面保持登录
- **头像自定义**：在完善信息页或账户弹窗中点击头像，支持拍照或从相册选择图片上传
- **账户信息**：侧边栏进入个人账户，查看手机号、昵称、注册时间、头像

### 数据持久化
- 学习数据（生词本、学习历史、花园状态）保存在浏览器 localStorage 中
- 用户账户信息（手机号、密码哈希、昵称、头像）保存在 MySQL 中

## 环境要求

- Python 3.10+
- MySQL 5.7+ / 8.0
- 现代浏览器（Chrome / Edge / Firefox 推荐）

## 本地运行

### 1. 配置 MySQL

确保本地 MySQL 服务已启动，并记住 root 密码。后端默认配置如下（如密码不同请修改 `server.py` 中的 `DB_CONFIG`）：

```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 3306,
    'user': 'root',
    'password': '123456',
    'charset': 'utf8mb4',
    'cursorclass': pymysql.cursors.DictCursor
}
```

首次启动后端时会自动创建 `rembord` 数据库和 `users` 表。

### 2. 启动后端

在项目根目录执行：

```bash
python server.py
```

服务启动后，终端会显示：

```
 * Running on http://127.0.0.1:5000
 * Running on http://172.x.x.x:5000
```

### 3. 访问应用

打开浏览器，访问：

```
http://localhost:5000
```

**注意**：必须使用上述地址访问，才能正常调用注册 / 登录 / 头像上传等后端接口。直接双击 `index.html` 或使用 VS Code Live Server 会导致 API 请求失败。

### 4. 注册与登录

1. 首次使用点击「立即注册」，填写手机号、密码、确认密码完成注册
2. 注册成功后进入「完善个人信息」页，可点击头像区域上传头像（支持拍照或相册选择），填写昵称
3. 已注册用户直接输入手机号和密码登录
4. 登录状态由 Session 保持，刷新页面无需重新登录

## 技术栈

### 前端
- HTML5 + CSS3
- 原生 JavaScript（ES Module）
- Web Speech API（语音合成）
- localStorage（学习数据持久化）

### 后端
- Python 3 + Flask
- PyMySQL（MySQL 连接）
- Werkzeug（密码哈希、文件上传安全）

## 浏览器兼容性

推荐使用 Chrome / Edge / Firefox 最新版，以确保语音播放、文件上传（capture）等功能正常工作。
