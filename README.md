# Rembord 背单词

一个基于浏览器的轻量级背单词应用，支持多种词库、专注模式、听写练习、能量花园等功能。

在线预览：https://cyf66667.github.io/rembord

## 功能特性

### 核心学习
- **多词库支持**：四级核心词汇、六级高频词汇、雅思红宝书、考研必背词汇
- **每日任务**：根据设置的背诵起止日期自动计算每日背诵量
- **单词卡片**：展示英文、音标、中文释义，支持一键标记（已认识 / 熟悉 / 不熟悉）
- **生词本**：自动按日期归档标记为"不熟悉"的单词，随时复习

### 专注模式
- 全屏沉浸式卡片背单词
- 点击卡片翻转查看中文释义
- 支持"认识 / 不认识"快速自评，不认识的词自动加入生词本

### 听写模式
- 系统语音朗读英文单词（使用 Web Speech API）
- 用户输入听到的单词，实时判断正误
- 支持从今日已学单词或任意词库中选题

### 默写测试
- 从今日已学单词 + 历史随机词汇中出题
- 支持两种模式：看英文写中文 / 看中文写英文
- 自我核对，不强制判分

### 能量花园
- 背单词获得水滴奖励
- 消耗水滴播种、浇水，培育植物成长
- 植物经历种子 -> 发芽 -> 幼苗 -> 开花四个阶段
- 开花后可收获，返还水滴并获额外奖励

### 学习日历
- 右上角日历图标查看学习记录
- 有学习记录的日期会显示绿色高亮
- 点击日期可查看当日学习的所有单词

### 数据持久化
- 所有学习数据（生词本、学习历史、花园状态、用户信息）保存在浏览器 localStorage 中
- 刷新页面或重新打开后数据不丢失

## 本地运行

本项目为纯前端项目，无需后端服务。

### 方式一：直接打开
用浏览器直接打开 `index.html` 即可使用（部分功能如词库加载可能需要本地服务器）。

### 方式二：本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 或使用 VS Code Live Server 插件
```

然后访问 http://localhost:8000

## 使用说明

1. 输入手机号登录（仅前端模拟，无真实验证）
2. 设置昵称，进入主界面
3. 在"学习"页选择词库和背诵起止日期，点击"打乱顺序并生成任务"
4. 在单词列表中对每个单词进行标记
5. 使用底部 Tab 切换听写模式、能量花园
6. 点击左上角 `☰` 打开侧边栏，查看生词本或学习日历

## 技术栈

- HTML5 + CSS3
- 原生 JavaScript（ES Module）
- Web Speech API（语音合成）
- localStorage（数据持久化）

## 浏览器兼容性

推荐使用 Chrome / Edge / Firefox 最新版，以确保语音播放等功能正常工作。
